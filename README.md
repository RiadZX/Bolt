# Project Cheetos
___
## A text editor and programming editor.

**Features:**
  * Open and save files.\n
  * Undo/Redo 
  * Copy/Paste/Cut commands
  * Run scripts in the editor
  * Help button for support
  * Encrypt/Decrypt selected text
